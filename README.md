# Cortex - AI Agent Platform for E-Commerce

A modern, responsive landing page for Cortex, an AI agent platform designed for e-commerce businesses.

## Features

- **Responsive Design**: Optimized for all screen sizes from mobile to desktop
- **Modern UI**: Clean, professional design with smooth animations
- **Interactive Elements**: Language selector, pricing toggle, FAQ accordion
- **Performance Optimized**: Fast loading with minimal dependencies

## Technologies Used

- HTML5
- CSS3 (with custom properties and modern layouts)
- Vanilla JavaScript
- Flaticon for icons
- Google Fonts (Geist)

## Live Demo

Visit the live site: [https://eshgin10.github.io/Cortex/](https://eshgin10.github.io/Cortex/)

## Local Development

1. Clone the repository
2. Open `index.html` in your browser
3. No build process required!
